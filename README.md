# Azure E-Commerce Lakehouse Project

## Overview

This project demonstrates an end-to-end modern data engineering platform using Azure cloud technologies.

The platform ingests batch and streaming e-commerce data, processes it through a medallion architecture (Bronze/Silver/Gold), and serves analytics dashboards through Snowflake and Power BI.
